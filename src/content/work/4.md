# VS Code: a launch-ready marketing site and docs gateway

Visual Studio Code needed a refreshed web experience that matched the polish of the editor. We focused on a clear value story, faster page loads, and tighter links into docs and marketplace content.

## Client goals

1. Clarify the product story for new users (why VS Code vs. other editors).
2. Reduce drop-off on download and onboarding flows.
3. Connect users to extensions, docs, and releases with fewer clicks.
4. Ship a maintainable design system to support weekly updates.

## What we delivered

1. **Narrative-first landing**  
   Hero, proof, and “choose your stack” sections that lead directly to tailored downloads.
2. **Extension highlights**  
   Dynamic cards pulling top extensions by category with quick-install links.
3. **Lightweight docs entry**  
   Curated “first 10 minutes” docs, keyboard shortcuts, and release notes surfaced on-page.
4. **Performance pass**  
   Image optimization, code-splitting, and a lean component library for faster TTI.

## Results

1. **+14% download completion** after simplifying the CTA path and adding stack presets.
2. **More engagement** with extensions: category cards drove more installs per visit.
3. **Faster publishing**: the new component library let the team ship weekly updates without design debt.

The VS Code site now tells a tighter story, moves users to action, and keeps them discovering the ecosystem.
